# Fetch API Sandbox

A responsive web application built to demonstrate practical usage of the **JavaScript Fetch API** for handling asynchronous HTTP requests, including fetching local resources and interacting with external REST APIs.

## Key Highlights
- Implemented GET requests for text, JSON, and external API data
- Implemented POST requests to submit data to a REST API
- Clean, responsive UI designed using Bootstrap 5
- Modular and readable JavaScript code structure

## Tech Stack
- HTML5
- CSS3 (Bootstrap 5)
- JavaScript (ES6, Fetch API)

## APIs & Data Sources
- Local resources: `sample.txt`, `users.json`
- External REST API: JSONPlaceholder (`https://jsonplaceholder.typicode.com`)

## Project Setup
1. Clone the repository
2. Run the project using a local development server
3. Open `index.html` in the browser

## Learning Outcome
This project demonstrates hands-on experience with:
- Asynchronous JavaScript
- HTTP methods (GET, POST)
- RESTful API interaction
- Frontend UI structuring and styling
